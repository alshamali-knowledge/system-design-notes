Welcome to my personal, production-grounded self-study curriculum. This live repository maps my journey from engineering fundamentals to mastering the deep prerequisites of high-level system architecture. 

Rather than memorizing standard high-level templates, this encyclopedia traces architectural trade-offs down to hardware physics, state mechanics, and verified production post-mortems.

> 📝 **Note on Content Generation & Curation:**
> The content within this encyclopedia is entirely generated by Qwen, utilizing its deep research capabilities for technical synthesis. My role consists of driving the curriculum's direction, editing the output, and organizing the material into this structured self-study platform.
> Some content diagrams are not ready yet, but there is a prompt for image generation for the missing diagram if needed

## 🛠️ Repository Structure

* **`docs/`** - Contains the raw Markdown files organized sequentially by chapter.
* **`mkdocs.yml`** - The configuration file for the live deployment site, styling, and navigation plugins.
* **`.github/workflows/deploy.yml`** - The automation script that updates the live tracking site every time a change is made.
